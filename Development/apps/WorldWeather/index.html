Built upon NASA World Wind, NASA World Weather is the first web application to ever display such a broad range of weather and other global spatial data viewed together in 3D.


Start by selecting the layers that you wish to view, in the order that you wish for them to be stacked on top of each other on the globe.

The same layers can be selected from either the "Available Layers" tab, in which data is organised according to its source, or from the "Layer Categories" tab, in which data is organised according to the type of information it displays.

Select the "Globe Options" tab in order to change the projection from the default 3D, or to select any country or city to navigate the globe to.

Select the "Legends" tab in order to view a legend for each of your selected layers, with information including the date and time of the particular layer.

Select the "Selected Layers" tab in order to view the layers that you have selected. There, you can click on each layer in order to hide or display it.

You can move the globe around by dragging the screen. You can also zoom in and out using your mouse or touchpad, or by using the controls on the bottom.

Hover over the globe in order to see the exact coordinates of that point displayed at the bottom of the screen.
